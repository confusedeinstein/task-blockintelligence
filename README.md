# Project Setup Guide

## Frontend Setup

1. Install dependencies:
   ```sh
   npm i
   ```
2. Build frontend:

   ```sh
   npm run build

   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Backend Setup

1. Set your MongoDB URI:

   ```sh
   mongodb://localhost:27017/
   ```

2. Install dependencies:

   ```sh
   npm i
   ```

3. Start the backend server:
   ```sh
   nodemon server.js
   ```
